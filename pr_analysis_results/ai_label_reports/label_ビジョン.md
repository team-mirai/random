### #221: 税制アルゴリズムに関する記述の修正

#### 説明

本提案は、匿名さんのご意見に基づき、「01_チームみらいのビジョン.md」の税制に関する記述を一部修正するものです。

**背景と目的:**
現行のマニフェスト案では、「変化に対応できる、しなやかな仕組みづくり」の一例として、「『所得』、『年齢』、『扶養』等の断片的な情報で作られた画一的な税制ではなく、個人の貯蓄額、景気動向、産業別動向などの情報に基づいた税制アルゴリズムを設計する」という記述があります。

この点について、匿名さんより「個人の貯蓄額」を税制アルゴリズムの考慮要素とすることについて懸念が示されました。主な理由は以下の通りです。

1.  **二重課税の懸念:** 所得に対して課税された後の資金が貯蓄となるため、貯蓄額に対してさらに課税することは二重課税にあたるのではないか、というご指摘です。これは、国民の公平感や税負担感に影響を与える可能性があります。
2.  **投資への阻害要因となる可能性:** 貯蓄への課税が、将来の投資に向けられるべき資金を減少させ、経済成長の足かせとなるのではないか、という懸念です。特に、本マニフェストがテクノロジー活用による成長を重視している点を踏まえると、投資マインドを冷え込ませる可能性のある施策は慎重に検討すべきです。
3.  **NISAなど現行政策との整合性:** 近年、政府はNISA（少額投資非課税制度）の普及などを通じて国民の貯蓄から投資へのシフトを奨励しています。そのような中で「貯蓄額を考慮した税制」を導入することは、政府の政策メッセージに矛盾を生じさせ、国民からの信頼を損なう恐れがあるとのご意見でした。貯蓄を推奨してきた経緯がある中で、貯蓄に対して課税するような印象を与えることは、国民の混乱を招きかねません。

これらの点を総合的に考慮し、より慎重な検討が必要な「個人の貯蓄額」という要素を現時点の記述からは削除し、「景気動向、産業別動向などの情報に基づいた税制アルゴリズムを設計する」と修正することで、より多くの国民の理解と共感を得られる政策案を目指すべきである、というのが匿名さんのご提案の趣旨です。

**変更点:**
「第二に変化に対応できる、しなやかな仕組みづくりです。」の項目内、税制に関する記述例を以下のように変更しました。

変更前:
* 『所得』、『年齢』、『扶養』等の断片的な情報で作られた画一的な税制ではなく、個人の貯蓄額、景気動向、産業別動向などの情報に基づいた税制アルゴリズムを設計する

変更後:
* 『所得』、『年齢』、『扶養』等の断片的な情報で作られた画一的な税制ではなく、景気動向、産業別動向などの情報に基づいた税制アルゴリズムを設計する

この変更により、税制アルゴリズムの検討要素から「個人の貯蓄額」を削除し、より多くの支持を得られる可能性のある記述としています。本マニフェストの目指す「テクノロジーで誰も取り残さない日本」を実現するための一環として、税制についても国民の意見を真摯に受け止め、より良い形を模索していく姿勢を示すものです。

この提案は、匿名さんのご意見を反映したものであり、チームみらいとして、より建設的な議論を喚起し、政策を磨き上げていくための一助となることを期待しています。皆様からのさらなるご意見をお待ちしております。

### #702: 福祉・介護分野におけるDX推進策を追加し、誰も取り残さないデジタル化を目指す

#### 説明

## 提案者

匿名ユーザー（杉原星さん）

## 変更の背景

今回の政策改善は、福祉・介護の現場で日々DX推進に取り組まれている杉原星さんの貴重なご意見に基づいています。現状の政策案「テクノロジーで誰も取り残さない日本をつくる」は、デジタル化推進の重要性を訴求していますが、特に高齢者やデジタル機器に不慣れな方々が直面する課題への具体的な対応策について、より踏み込んだ記述が求められていました。

杉原さんからは、以下のような現場の課題が共有されました。

*   **デジタルデバイドの深刻さ**: スマートフォンやPCの操作に慣れていない高齢者や一部の職員にとって、新しいシステムの導入が大きな障壁となっている。
*   **アナログ業務の根深さ**: 未だに紙媒体での手書き業務が多く、非効率な作業が現場の負担を増大させている。
*   **事業者間連携の壁**: 複数の事業所が関わるケースにおいて、情報共有システムが標準化されていないため、サービス連携がスムーズに行えず、利用者の方々へのケアの質にも影響が出ている。
*   **変化への抵抗感**: 長年慣れ親しんだ業務プロセスを変えることへの心理的な抵抗感が、DX推進の足かせとなっている場合がある。

これらの課題は、福祉・介護分野におけるDXが単なる技術導入に留まらず、利用者と提供者の双方にとって真に有益なものとなるために、きめ細やかな配慮と戦略的なアプローチが必要であることを示しています。

## 変更の目的

本提案の主な目的は、政策マニフェスト「01_チームみらいのビジョン.md」において、福祉・介護分野における「誰も取り残さないDX」を具体的に推進するための施策を明記することです。

具体的には、以下の達成を目指します。

1.  **情報連携の強化と業務効率化**: 事業者間の情報共有システムを標準化することで、アナログ業務を削減し、職員の負担を軽減する。これにより、職員がより質の高いケアに集中できる環境を整備する。
2.  **デジタルアクセシビリティの向上**: デジタル機器に不慣れな職員や利用者に対して、丁寧な操作研修の機会を提供する。また、音声入力などの代替入力手段の導入を支援し、誰もがテクノロジーの恩恵を受けられるようにする。
3.  **ケアの質の向上**: 効率化によって生まれた時間やリソースを、利用者一人ひとりに向き合う時間に充てることで、ケア全体の質を高める。
4.  **変化へのハードル低減**: 行政からの適切な働きかけと、具体的なサポート策をセットで提示することにより、現場の変化への抵抗感を和らげ、DXへの移行をスムーズにする。

### #925: コストの全体最適化

#### 説明

本プルリクエストは、Yuichi Furukawa様からのご提案に基づき、「01_チームみらいのビジョン.md」の政策文書に対して一部修正を行うものです。「デジタル時代の当たり前をやりきる」という項目において、より具体的な政策を提示し、現状の課題に対応することを目的としています。

**1. 変更の概要**

政策文書の「デジタル時代の当たり前をやりきる」の章にある「IT/AIによって行政手続きを民間サービス並に改善させ、行かせない、書かせない、待たせない、迷わせない行政を実現」という記述の直後に、以下の太字の一文を追加しました。

「**特に、全国の自治体で共通して必要となるサービスについては、国が標準システムを開発・提供し、全体的なコスト最適化と質の均一化を目指します。**」

**2. 提案の背景と課題認識**

現在の日本の地方自治体におけるDX（デジタルトランスフォーメーション）は、多くの課題を抱えています。特に、政府が推進する標準準拠システム（いわゆるガバメントクラウド）への移行に関しては、その理想とは裏腹に、現場の自治体にとってはかえって負担増やコスト増を招いているケースが散見されます。これは、標準仕様の解釈がベンダーごとに異なったり、既存システムとの連携が複雑化したりすることなどが要因として挙げられます。結果として、限りある税金が効果的に活用されているとは言い難い状況が生じており、この点は緊急に是正されるべき喫緊の課題であると認識しています。

また、AI（人工知能）のような先進技術を行政サービスに導入し、その恩恵を国民生活隅々まで行き渡らせることは非常に重要です。しかし、その前提となる自治体の情報システム基盤に目を向けると、いまだにメインフレーム時代を彷彿とさせるような古いユーザーインターフェース（UI）が維持されていたり、いわゆる「地場ベンダー」と呼ばれる事業者の技術力やコスト意識が時代に合わせて十分にアップデートされていなかったりする実情があります。このような状況では、最新技術の導入効果も限定的となり、税金の浪費に繋がりかねません。まさに「絵に描いた餅」で終わってしまう危険性があります。

**3. 変更の目的と期待される効果**

今回の変更の中心的な目的は、全国の自治体で共通して必要不可欠となる行政サービス（例えば、住民記録、税務、福祉関連など）について、国（具体的にはデジタル庁のような専門組織を想定）が主体となって標準化されたシステムを開発し、各自治体に提供することです。これにより、以下の効果を期待します。

*   **全体的なコストの最適化:** 各自治体が個別にシステムを開発・調達するのではなく、国が一括して開発・提供することで、重複投資を避け、規模の経済を活かした大幅なコスト効率の改善を目指します。ここで用いた「削減」ではなく「最適化」という言葉には、単に費用を減らすことだけを目的とするのではなく、提供されるサービスの質を維持・向上させつつ、最も効率的なリソース配分を実現するという意図が込められています。
*   **質の均一化と向上:** 国が提供する標準システムは、最新の技術動向、セキュリティ基準、そして何よりも利用者である住民や自治体職員にとっての使いやすさ（UI/UX）を考慮して設計されるべきです。これにより、どの自治体においても一定水準以上の質の高い行政サービスが提供されることを目指します。また、地場ベンダーの技術力に左右されることなく、全国どこでも質の高いサービスが受けられる環境を実現します。
*   **自治体職員のDX推進と業務効率化:** 標準化され、使いやすいシステムが提供されることで、自治体職員のITリテラシー向上やDX（デジタルトランスフォーメーション）意識の醸成が期待されます。手作業や紙ベースの業務が大幅に削減され、より創造的で付加価値の高い業務に集中できるようになることを後押しします。これは「行かせない、書かせない、待たせない、迷わせない行政」の実現に不可欠です。
*   **「仕組み」ではなく「具体的なWebサービス」の提供:** 本提案は、抽象的なガイドラインや方針を示すだけでなく、実際に「使える」具体的なWebサービスとして提供されることを重視しています。これにより、自治体はシステム選定やカスタマイズの負担から解放され、迅速にサービスを導入・活用できるようになります。

**4. 本マニフェストにおける位置づけ**

**背景と目的:**

わが国では高齢化が急速に進んでおり、デジタル化の恩恵が一部の人々に偏ることなく、全ての世代に行き渡ることが「誰も取り残さない」社会を実現する上で極めて重要です。しかし現状では、操作の難しさや専門用語の壁などにより、多くの高齢者の方々がデジタル技術の活用に困難を感じています。

# 提案の意図と目的
現代社会において多くの人々が感じる「生きづらさ」の根源の一つに、自らの幸福を自由に定義し追求することの困難さがあるという問題意識に基づいています。特に、これから生まれてくる子どもたちには同様の感覚を抱かせたくないという強い思いから、この理念をチームみらいの根幹的な価値観として明確に位置づけることを目的としています。

この「幸せを定義する自由」は、単に個人的な欲望の追求を意味するのではなく、他者の同様の自由を尊重し、互いの幸福が調和し、社会全体としてより大きな幸福（シナジー）を生み出す形で実現されるべきであるという点が重要です。この考えは、個人の尊厳と自己実現を核としつつ、他者との共生や社会全体の調和を志向するものです。

この理念をマニフェストの基本方針として掲げることで、チームみらいが目指す社会像をより明確にし、あらゆる政策立案・実行の指針とすることを目指します。また、この理念はチームみらいが既に掲げている「私たちは誰かを貶めない」「私たちは分断を煽らない」「私たちは何も決めつけない」といった多様性を尊重する価値観とも強く結びつき、それらを補強し、より深めるものと考えられます。

# 背景
この提案の背景には、FLOW氏による以下のnote記事における詳細な考察があります。この記事では、哲学的・心理学的視点、さらには記号創発システム論を交えながら、「自分にとっての幸せを定義する自由」の重要性、現代社会における生きづらさの根源、そしてその解放に向けたアプローチが探求されています。

参考資料：自分にとっての幸せを定義する自由：生きづらさの根源とその解放
[https://note.com/flow_theory/n/nc7cd88f79e3b](https://note.com/flow_theory/n/nc7cd88f79e3b)

2.  **国民の政治参画意識の向上:** 自分たちの意見が政策に反映される実感を国民が得やすくなり、政治への関心と信頼が高まります。
3.  **説明責任の明確化:** 政府や行政機関は、自らの政策判断の根拠をデータで示す必要性が高まり、説明責任がより明確になります。
4.  **リソース配分の最適化:** 国内のどの分野に重点的に投資すべきか、あるいは国際社会にどの程度の貢献が可能かといった判断について、国民全体のコンセンサス形成が容易になります。例えば、佐野氏が例として挙げられた「2人の子供とりんごの分配」のように、現状のリソース量を正確に把握することで、無理のない、かつ効果的なリソース配分が可能となります。
5.  **危機管理能力の強化:** 災害時や経済危機など、有事の際には、国民全体がリソース状況を共有することで、パニックを防ぎ、協力体制を築きやすくなります。

**背景**

この提案は、東堂暁光山氏の「あらゆるリソースを可視化し、そのうちのいくつ分配できるかという意見をブロードリスニングで集約すれば健全かつ長期安定した国家を維持することが可能」という問題意識に基づいています。

デジタル技術、特にデータ可視化技術やAIによる分析技術の進展は、この課題を解決する大きな可能性を秘めています。国民がまるで自社の経営状況を把握するように、自国のリソース状況を直観的に理解できるインターフェースを提供することで、より多くの人々が「自分ごと」として国の将来を考え、積極的に政策議論に参加する未来を目指します。

この変更は、マニフェストが掲げる「ブロードリスニングの技術を使い継続的に解像度高く国民の意見・アイデアを集めることができるデジタル民主主義の実践」という方針を具体化するものであり、テクノロジーを活用して国民と政治の距離を縮め、より開かれた、信頼される政府を実現するための重要な一歩となると確信しています。

### #1742: UI/UX改善：誰もがテクノロジーを活用できるためのサポート体制を明記

#### 説明

匿名ユーザーさんからの提案に基づき、政策案を更新しました。

現状の政策案では、AIをはじめとするテクノロジー活用の重要性が随所で述べられていますが、それを「誰でも」利用できるようにするための具体的なサポート体制については言及が十分ではありませんでした。

そこで、「デジタル時代の当たり前をやりきる」のセクションに、以下の内容を明記しました。

1.  **アクセシビリティの抜本的向上：** 専門知識を持たない人や、新しい技術に不慣れな人でも、直感的なマニュアル、分かりやすい動画解説、AIアシスタント、そして専門チームによる個別サポートを通じて、安心してテクノロジーを利用できるようになります。これにより、**エンジニアや技術に明るい層と、そうでない利用者との間に生じがちな情報格差・利用格差を解消します。**
2.  **利用促進と格差解消を通じた公正な社会の実現：** テクノロジーの「使い方」に関する手厚いサポートを提供することで、利用のハードルを劇的に下げ、より多くの人々がテクノロジーの恩恵を受けられるようにします。これは、経済的な機会均等だけでなく、教育、医療、行政サービスへのアクセス向上にも繋がり、**新たな技術がもたらす便益から誰も取り残されることのない、より公正な社会の実現**に貢献します。
3.  **真のカスタマーサクセスの実現：** 利用者一人ひとりがテクノロジーを効果的に活用し、それぞれの目的を達成できるよう能動的に支援する「カスタマーサクセス」の考え方を政策の中心に据えます。単に技術を提供する側の一方的な情報提供に留まらず、利用者の成功体験を共に創り上げることを目指します。
4.  **技術革新と社会的包摂の両立、そして政策への信頼性向上：** 「誰も取り残さない」という理念を、具体的な行動指針とサポート体制の明記によって裏打ちすることで、政策全体の信頼性が高まります。これにより、**日本が革新的な技術開発を積極的に推進しつつも、その恩恵が全国民に行き渡る、バランスの取れた発展モデル**を世界に示せるようになると考えます。

### #1760: 政策の基本理念を明確化：冒頭に「めざす社会の基本理念」を追加

#### 説明

この提案は、しろなさんとの議論に基づき、「01_チームみらいのビジョン.md」の冒頭部分を修正し、政策全体の根底にある「めざす社会の基本理念」をより明確に示すことを目的としています。

特に、「誰も取り残さない」というこの政策のコアメッセージは、まさにしろなさんが指摘された「パレート最適の追求」や「ナッシュ均衡下での調和」という国家の役割に通じるものです。テクノロジーを活用することで、これまで実現が難しかった個々人の事情に配慮したきめ細やかな対応や、多様な価値観が共存できる社会システムのデザインが可能になるという期待も、この基本理念には込められています。
