## 変更の概要
本提案は、政策文書「４．医療」の「ビジョン」セクションに、国民の主体的な健康増進への取り組みを奨励し、その努力が報われる仕組みづくりを目指す旨の一文を追加するものです。

追加された文言：
「人生100年社会において、健やかに楽しく人生を送れるために、国民一人ひとりが主体的に健康増進に取り組むことを奨励し、その努力が報われるような仕組みづくりも目指します。」

## 変更の意図と目的
この変更の主な意図は、医療政策の基本的な考え方として、個人の健康に対する意識と行動変容を積極的に支援する姿勢を明確にすることです。人生100年時代と言われる現代において、国民一人ひとりが生涯を通じて健やかで質の高い生活を送るためには、治療中心の医療だけでなく、病気になりにくい身体づくりや健康的な生活習慣の確立が不可欠です。
本提案は、そのような主体的な健康増進活動が社会的に評価され、かつ個人にとってもメリットのある形（例えば、議論のあった保険料インセンティブなど、具体的な施策は今後の検討課題）で報われるような制度設計の方向性を示すことを目的としています。これにより、国民の健康寿命の延伸、生活の質の向上、そして将来的には医療費の適正化にも貢献することを目指します。

## 変更の背景
日本の医療制度は国民皆保険という優れた基盤を持っていますが、少子高齢化の急速な進展や医療技術の高度化に伴い、社会保障費は増大の一途をたどっています。現行の政策案でもこの課題認識は共有されており、医療制度の持続可能性を確保するための多角的なアプローチが模索されています。
今回の追記は、ユーザー「無名」様との対話の中で、「疾病予防や健康増進にもっとインセンティブがあって良いのではないか」「例えば、一定期間医療機関にかからなかった場合に保険料が割引されるような仕組みは考えられないか」といったご意見をいただいたことがきっかけとなっています。このご意見は、現行の「ビジョン」にある「テクノロジーとデータを使い、より緻密なシステムを構築することで医療の非効率を減らし、より費用対効果の高い医療が報われる制度を目指します」という方針とも親和性が高く、個人の行動変容を促すという観点から、政策全体の方向性を補強するものと考えられます。
治療が必要な方への医療アクセスを確保しつつ、健康な方々や軽微な症状の方々が、予防やセルフケアにより積極的に関与することの重要性を政策ビジョンに明記することで、よりバランスの取れた持続可能な医療システムの構築に資すると考えられます。

## 期待される効果
この一文をビジョンに加えることで、以下のような効果が期待されます。
1.  **国民の健康意識の向上:** 政策として健康増進が奨励されることで、国民一人ひとりが自身の健康についてより深く考え、主体的に取り組むきっかけとなります。
2.  **予防医療の推進:** 健康増進への努力が報われる仕組みの検討を促すことで、病気の早期発見・早期治療に加えて、病気にならないための予防活動が社会全体で推進されます。
3.  **医療費の適正化:** 長期的に見て、国民の健康状態が改善されれば、疾病の発生率や重症化率が低下し、結果として医療費の伸びの抑制につながる可能性があります。

**背景と現状の課題**

新型コロナウイルス感染症（COVID-19）のパンデミックは、感染症の流行状況をリアルタイムで把握することの重要性を浮き彫りにしました。国民は、感染状況が不透明な中で日々の生活や経済活動のめりはりを付けることに困難を感じてきました。

現在、主な感染症の流行状況把握手段として、以下のものが挙げられます。

1.  **定点医療機関からの報告:** 主に急性呼吸器感染症の把握に用いられていますが、いくつかの課題があります。
    *   **代表性の問題:** 2020年のパンデミック初期と比較して、症状があっても医療機関を受診する人や検査を受ける人が減少する傾向にあり、報告される数が必ずしも全体の流行状況を正確に反映しているとは言えなくなっています。受診行動の変化が統計の連続性や解釈の妥当性に影響を与えています。
    *   **タイムラグ:** 報告が週単位など、一定の時間を要するため、最新状況の把握という点では遅れが生じます。

2.  **下水サーベイランス:** ウイルス量などを下水から検出する方法で、症状の有無や受診行動に左右されにくく、実際の市中感染状況を比較的よく反映すると期待されています。しかし、現状では以下のような課題があります。
    *   **リアルタイム性の欠如:** 結果の判明までに数週間を要することが多く、流行の初期段階での迅速な対応や、日々の行動変容を促す情報としては即時性に欠けます。

これらの課題を踏まえ、より迅速かつ実態に近い流行状況の把握を可能とする新たなシステムの構築が急務であると考えられます。

**提案する政策概要**

1.  **下水サーベイランスの迅速化:**

**主な改善点:**
政策概要の冒頭部分に、本制度における「成果」の定義を具体的に記述しました。これまでは「どれだけ良くなったか」という表現に留まっていましたが、これでは治癒や機能回復のみが「成果」と捉えられ、多様な患者さんの状態やニーズに対応できない可能性が議論されました。
そこで、「成果」とは、単に疾患の治癒や機能回復のみを指すのではなく、患者さんのQOL（生活の質）の維持・向上、症状の緩和や進行抑制、治療に対する納得感なども含めた多角的な視点から評価されるものであることを明記しました。

さらに、どのような状態が「成果」と見なされるのか、また逆に「成果」とは見なされないのかについて、具体的な例を追記しました。

*   **「成果」であるものの具体例として:**
    *   慢性疾患のコントロール状態の改善（血糖値や血圧の安定など）
    *   リハビリテーションによる日常生活動作（ADL）の向上
    *   緩和ケアにおける疼痛の軽減
    *   難病や障害を持つ方々の状態維持やQOL向上
*   **「成果」とは見なされないものの具体例として:**
    *   科学的根拠に乏しい、または患者さんのQOL向上に繋がらない治療の漫然とした継続
    *   医学的必要性の明確でない過剰な検査や投薬
    *   患者さんの明確な意思に反するような医療介入

**改善の意図・目的:**
この改善の主な目的は、成果連動型報酬制度の導入において、その評価軸となる「成果」の定義を明確にすることで、制度の意図しない解釈や運用を防ぐことです。
特に、治癒や機能回復が主な治療目標とならない慢性疾患の患者さん、難病や障害をお持ちの方々、緩和ケアを必要とする方々など、多様な状況にある人々が不利益を被ることなく、質の高い医療を継続して受けられるようにする意図があります。
また、患者さん一人ひとりの価値観や生活の質を尊重した、より個別化された医療の実現を目指すという政策全体の方向性を、この部分でも明確に示したいと考えました。
「成果ではない」具体例をあえて示すことで、医療の質の観点から問題となる可能性のある行為を抑制し、より適切で効率的な医療資源の配分を促すことも意図しています。

+    * 2022年度に242の急性期病院を対象に行われた調査では30種を超える診療行為を健康成果に直結しない医療と定義し、それらによる医療費が同施設の医療費の0.23〜0.51％を占めるとされました。これは、同期間の医療費に単純換算すると約3,000〜7,000億円規模に達すると試算されています。  
+    * 現在、国民医療費は2022年度に46兆円を超えましたが、少子高齢化と高度医療の進展により今後も増えていくことが予想されています。そのため、医療費全体を抑制する施策とともに医療の成果にも着目した財源配分の検討が必要です。
+
+    ### 政策概要
+
+    * 診療行為のエビデンス、費用対効果や重症度に基づく自己負担割合の複数段階化を検討します  
+      * 現在の健康保険法や高齢者医療確保法では、年齢による自己負担割合が言及されていますが、医療価値に応じた負担区分の段階化が可能となるよう、これらの法改正を検討します  
+      * 現在の診療報酬システムでは、医療機関が審査支払機関に請求する診療報酬のみが設定されていますが、診療内容による自己負担割合の影響についても制度に組み込みます  
+      * 上記の緻密な診療報酬制度を構築するため、全国医療データベース（NDB）に蓄えられた診療実績を解析し、柔軟に機動的に診療行為の有効性、費用対効果について議論するための基盤を整備します  
+    * 制度の複雑化による医療の現場や審査機関での事務処理の複雑化を解消するための仕組みの開発および導入を支援します  
+      * 診療報酬制度の複雑化により、医療機関の窓口での計算もより複雑になります  
+      * 医療機関への負担、サービスの質の低下を避けるために、電子レセプトに連携し窓口で即時に自己負担額が算出できるAI、システムの開発を支援します  
+      * 併せて、医療DXを推進する支援の枠組みについても整備します  
+    * これらの仕組みにより、医療の質、家計への負担を維持したまま、医療財政への負担軽減を目指します。実際にフランスでは医薬品について医療上の有用性評価を行い、自己負担割合が設定されるという仕組みが導入されています（国民医療費をxx-xx％減らしたという実績もあります）  
+      * フランスでは医薬品ごとに有効性や副作用、代替治療の有無などの位置づけ、対象となる疾患の重篤度、公衆衛生上の意義などで区分されます  
+      * この評価に基づいて、外来用医薬品の自己負担割合が段階的に設定されます  
+      * 評価が不十分な医薬品は自己負担割合が高くなることもありますが、一方で、抗がん剤など代替の効かない医薬品については自己負担割合を0%としています  
+      * この他にも、海外の医療制度では年齢、所得以外に診療の内容や受診経路などで自己負担割合を変化させている例があります

+    ## 2）治療成果に報いる医療アウトカム評価制度の導入
+
+    ### 現状認識・課題分析
+
+    * 日本では医療費の約3割が、糖尿病・COPD・心不全などの慢性疾患に費やされています  
+    * 日本では慢性疾患のアウトカム指標（HbA1c、FEV1、BNPなど）については医学的に整理がなされ、学会を中心にガイドラインの整備が進められているものの、医療制度の中で医療の質の向上を促す仕組みが十分に設計されているとは言いがたい側面がある  
+      * 現行の医療制度は行われた処置・診察の回数に応じて診療報酬が設計されており、疾患コントロールが改善したことに対するインセンティブは設計されていない  
+      * 糖尿病や高血圧、脂質異常症などの慢性疾患の多くは重症化し、虚血性心疾患などのイベントが発生するまで、自覚症状に乏しいことも多く、患者サイドにも治療を強化し、積極的に状態をよくしようと動機づけしにくい場合がある  
+    * 海外では、治療成果に応じて報酬が支払われる制度が導入され、一定の課題が示されつつも医療の最適化と費用削減を両立が期待されています
+
+    ### 政策概要
+
+    * 成果評価においては、患者一人ひとりの状態や疾患の特性を十分に考慮し、改善だけでなく、症状の安定や生活の質の維持・向上も重要な評価軸とします。特に、体質的な要因や難病など、改善が困難な疾患については、画一的な成果指標を適用するのではなく、個別最適な医療を引き続き提供できるよう配慮します。
+    * 「どれだけ診療が行われたか」だけでなく、「どれだけ良くなったか」にも報いる医療制度への転換を目指し、成果連動型の診療報酬制度を導入する  
+      * 糖尿病、COPD、心不全など、アウトカム指標が整備され患者数・医療費規模の大きい慢性疾患の治療成果を「血糖値」「呼吸機能」「心不全マーカー」などの指標で評価  
+      * 治療の効果が高い医療機関に対しての報酬加算、患者に対しての還元を設計し、医療機関・患者の双方に動機づけを行う  
+    * 診療データを匿名化し、全国医療データベース（NDB）で一元管理。AIを活用した公平で迅速な評価を行う仕組みを整えます  
+      * アウトカム評価の効率化と医療現場の負担軽減のため、電子カルテの標準化、相互運用性の確保を進めます  
+      * 診療記録や検査結果をデータ化し、治療成果を判定、成果加算が自動的に反映される仕組みを設計します  
+      * アウトカム評価指標の提出、確認が行われることで医療機関の負担が増えないよう、民間企業と連携した電子カルテの解析システムの開発を促します  
+      * また、電子カルテの情報連携においてセキュリティとプライバシーを確保するため、必要な法令、医療機関向け指針の整備を進めます
