#### Issue内容

本提案は、小山さんによる「チームみらい マニフェスト」への重要な追加提案です。近年拡大するギグワークやフリーランスといった新しい働き方に関して、その担い手が不当に低い報酬によって経済的困窮に陥ることを防止し、適正な報酬水準を保障するとともに、能力向上と持続可能な働き方を支援することを目的とした「ギグワーク・フリーランス等の適正報酬保障に関する法案（仮称）」を新たに策定し、マニフェストに加えるものです。

背景として、ギグワーカーやフリーランスは、柔軟な働き方が可能である一方、発注者との力関係の不均衡から、不当に低い報酬、契約内容の不明確さ、報酬の未払いや一方的な減額といった問題に直面しやすい状況があります。現行法ではこれらの新しい働き方を十分に保護できておらず、働き手の生活不安定化やスキルアップ機会の喪失を招き、経済全体の活力低下にも繋がる懸念があります。

この法案では、主に以下の点を定めています。
1.  **総則**: 法の目的と、「特定受託者」「特定発注者」「プラットフォーム事業者」「最低参照報酬」等の定義を明確化。最低参照報酬は強制的な最低賃金ではなく、不当性判断の参照値と位置づけます。
2.  **最低参照報酬の設定と適正化の推進**: 業務の性質、難易度、専門性、経費等を考慮した報酬設定の原則、厚生労働省への「特定受託者報酬算定委員会（仮称）」設置、同委員会による最低参照報酬の算定とガイドライン公表、報酬算定根拠の開示義務などを規定。
3.  **特定発注者およびプラットフォーム事業者の義務と協力**: 最低参照報酬を著しく下回る報酬での発注禁止、プラットフォーム事業者による不当な低報酬案件への警告・修正勧告・掲載拒否等の措置、相談窓口設置、紛争解決支援体制整備、スキルアップ支援の努力義務などを規定。
4.  **特定受託者の保護、紛争解決、研修等**: 国による匿名通報窓口設置、紛争解決支援制度、特定受託者・発注者双方への研修機会提供などを規定。
5.  **その他**: 関連社会保障制度との連携、罰則規定（行政指導、勧告、企業名公表、過料、悪質な場合の罰金刑等）を設けています。

この提案により、特定受託者の経済的安定、公正な取引慣行の確立、多様な働き方の健全な発展、スキルアップ支援、紛争予防・解決が期待されます。

+# ギグワーク・フリーランス等の適正報酬保障に関する法案（仮称）
+
+# 第1章 総則
+
+# 第1条（目的）
+# この法律は、特定の事業者と業務委託契約を締結し、またはプラットフォームを介して業務を請け負う個人（以下「特定受託者」という。）が、その業務に対する不当に低い報酬によって経済的困窮に陥ることを防止し、**適正な報酬水準を保障する**とともに、**能力向上と持続可能な働き方を支援する**ことで、特定受託者の生活の安定と多様な働き方の健全な発展に寄与することを目的とする。
+
+# 第2条（定義）
+# 1. 特定受託者:
+#    a. 特定の事業者に雇用されず、自己の裁量と責任において業務を請け負い、報酬を得る個人を指す。
+#    b. クラウドソーシングプラットフォーム等のオンラインサービスを介して業務を受託する者を含む。
+#    c. 前項の定義にかかわらず、**年間所得、業務時間、または特定の業務形態等の基準を政令で定め、本法の適用対象者を明確化する**。
+# 2. 特定発注者:
+#    a. 特定受託者に業務を発注する法人または個人を指す。
+#    b. 特定受託者と業務委託契約を締結する事業者、およびプラットフォーム上で業務を発注する者を含む。
+# 3. プラットフォーム事業者:
+#    a. 特定受託者と特定発注者間の業務委託契約の媒介を行うオンラインサービスを提供する事業者を指す。
+# 4. 最低参照報酬:
+#    a. 特定受託者に支払われるべき業務の対価として、本法に基づき算定される参照基準となる金額。
+#    b. 強制的な「最低賃金」ではなく、**特定受託者の生活費、経費、専門性等を考慮した不当性の判断基準となる「参照値」**として機能する。
+
+# 第2章 最低参照報酬の設定と適正化の推進
+
+# 第3条（最低参照報酬の原則）
+# 1. 特定受託者に支払われる報酬は、その業務の性質、難易度、専門性、所要時間、市場相場、および業務遂行に要する経費（通信費、機材費、消耗品費、ソフトウェア利用料、交通費等）を総合的に勘案し、**不当に低いものであってはならない**。
+# 2. 特に、**業務に要する時間（合理的かつ標準的な所要時間。これには準備時間、待機時間等も含む）を換算した場合**、地域別最低賃金法に定める賃金水準を著しく下回る報酬は、不当な低報酬とみなされるものとする。
+# 3. 特定発注者は、特定受託者の専門性や経験を尊重し、適正な報酬水準を維持するよう努めなければならない。
+
+# 第4条（最低参照報酬額の算定委員会）
+# 1. 厚生労働省に「特定受託者報酬算定委員会」（仮称）を設置する。
+# 2. 本委員会は、特定受託者、特定発注者、プラットフォーム事業者、学識経験者、および労働行政関係者で構成し、**関係省庁の代表者を加える**。
+# 3. 本委員会は、業種、業務内容、地域、**専門性のレベル、成果物の市場価値**等を考慮した上で、定期的に各業務カテゴリにおける「最低参照報酬」の目安を算定し、**具体的なガイドラインとして**公表する。
+# 4. 算定にあたっては、関連業界団体、特定受託者団体等からの意見聴取を義務付けるとともに、**国民一般からの意見募集も行う**。
+
+# 第5条（報酬の透明化と説明責任）
+# 1. 特定発注者は、特定受託者に対し、業務委託契約を締結するに際し、報酬の金額、**その算定根拠（例: 想定作業時間とその単価、成果物単価、専門性に応じた加算、諸経費の考慮など）**、支払い時期、支払い方法を明確に書面または電磁的方法で開示しなければならない。
+# 2. 特定受託者が報酬の算定根拠について説明を求めた場合、特定発注者は正当な理由なくこれを拒否してはならない。
+
+# 第3章 特定発注者およびプラットフォーム事業者の義務と協力
+
+# 第6条（不当な報酬の禁止）
+# 1. 特定発注者は、第4条で算定された最低参照報酬を著しく下回る報酬で特定受託者に業務を発注してはならない。
+# 2. **フリーランス保護新法第12条（特定受託事業者の事業活動に不当な影響を及ぼす行為の禁止）の規定を遵守するとともに、本法における「不当な報酬」の判断基準は、政令で具体的に定める**。
+# 3. 特定発注者は、報酬支払い遅延、一方的な報酬減額、不当な返品・キャンセル等、特定受託者の経済的利益を不当に害する行為を行ってはならない。
+
+# 第7条（プラットフォーム事業者の義務）
+# 1. プラットフォーム事業者は、そのプラットフォーム上で発注される業務の報酬が不当に低くならないよう、適切なシステム構築と監視を行う義務を負う。
+# 2. 第4条で算定された最低参照報酬を著しく下回る案件が投稿された場合、発注者に対し**自動的な警告表示、修正勧告、または掲載拒否、アカウント一時停止等の措置を講じる義務**を負う。
+# 3. プラットフォーム事業者は、特定受託者からの報酬に関する相談窓口を設置し、**プライバシーに配慮した上で**、必要に応じて紛争解決支援を行う体制を整備しなければならない。
+# 4. 報酬の未払いや一方的な減額がプラットフォーム上で発生した場合、その対応方針を明確化し、特定受託者への情報提供を行うとともに、**一定の条件下で報酬の立替払いや保証を行う制度の導入に努める**。
+# 5. プラットフォーム事業者は、特定受託者が自身のスキルや市場価値を適切に評価し、適正な報酬を得られるよう、**スキルアップや単価交渉に関する情報提供、研修機会の案内**を行う努力義務を負う。
+
+# 第4章 特定受託者の保護、紛争解決、研修等
+
+# 第8条（相談・通報体制）
+# 1. 厚生労働省および関係省庁は、特定受託者が不当な報酬に関する相談や通報を**匿名でも**行える窓口を設置する。
+# 2. 通報内容に基づき、必要に応じて特定発注者やプラットフォーム事業者に対する調査、指導、勧告、**改善命令**を行うことができる。
+
+# 第9条（紛争解決の支援）
+# 1. 不当な報酬に関する紛争が発生した場合、国は特定受託者と特定発注者間のあっせん、調停、または仲裁による解決を支援する制度を設ける。**これらの手続きは、特定受託者の負担を軽減するため、迅速かつ簡易に行われるものとする**。
+# 2. プラットフォーム事業者は、そのプラットフォーム内で発生した報酬に関する紛争解決に積極的に協力しなければならない。
+# 3. 弁護士会や司法書士会等の関係団体と連携し、特定受託者への法的支援を強化する。
+
+# 第10条（研修・教育の推進）
+# 1. 国および地方公共団体は、特定受託者が自身のスキルを向上させ、市場価値を高めるための研修や情報提供を推進する。
+# 2. 特定受託者に対し、適正な報酬設定、契約交渉、経費管理、税務に関する知識習得のための教育機会を提供する。
+# 3. 特定発注者に対し、特定受託者との適正な取引関係を構築するための研修や啓発活動を行う。
+
+# 第5章 その他
+
+# 第11条（関連社会保障制度との連携）
+# 国は、特定受託者の報酬の適正化に加え、**フリーランスの社会保障制度（健康保険、年金、労災保険等）への円滑な加入や給付の充実**について、別途検討を行い、必要な措置を講ずるものとする。
+
+# 第12条（罰則）
+# 第6条（不当な報酬の禁止）に違反し、是正勧告に従わない特定発注者、または第7条（プラットフォーム事業者の義務）に定める義務を著しく怠り、改善命令に従わないプラットフォーム事業者に対しては、行政指導、勧告、企業名の公表、過料、**さらに悪質な場合には事業活動の一部制限や、法人の代表者等に対する罰金刑等の罰則**を科すことができる。
+
+# 附則
+# 1. （施行期日）本法は、公布の日から起算して〇年〇月〇日を超えない範囲において政令で定める日から施行する。
+# 2. （経過措置）本法施行前に締結された契約については、その契約期間が終了するまでの間、本法の一部規定の適用について政令で定める経過措置を設けることができる。
+# 3. （関係法令の改正）必要に応じて、労働基準法、最低賃金法、フリーランス保護新法その他の関係法令の改正を行うものとする。
